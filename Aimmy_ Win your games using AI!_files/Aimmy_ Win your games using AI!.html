Aimly is not designed to be suitable for games with aggressive anti-cheats like Valorant, Fortnite (HWID Ban), Apex Legends, or Warzone. We are not responsible for any bans that occur due to Aimly; you assume all responsibility when using our product.

Aimly is a passionate community project. We process data locally via your GPU and use free services like GitHub for hosting. Because our operating costs are incredibly low, we don't need to ruin the experience with an ad-supported model.

Ensure you have the x64 versions of both the .NET 8.0 Desktop Runtime and Visual C++ Redistributables installed. If issues persist, please drop by our Discord server for instant help!
